# PocketCart 🛒✨

Welcome to **PocketCart**, a browser extension + MERN dashboard that helps you save products from anywhere on the internet into one cute, searchable wishlist.

No more scattered wishlists, random screenshots, endless tabs, or “wait, where did I see that?” moments. Just click the extension, save the product, and find it later in your PocketCart dashboard.

## 🌐 What It Does

PocketCart lets you save products while browsing online.

The extension captures:

- product/page title
- URL
- website source
- product image
- notes
- collection

Then the dashboard lets you:

- search saved products
- filter by collection or source
- edit product details
- delete products
- open the original product page

## 🧃 Why I Built This

I wanted to build something more useful than a basic todo app while learning the MERN stack.

PocketCart helped me understand how a real full-stack project works:

```txt
Chrome Extension -> Express API -> MongoDB -> React Dashboard
```
